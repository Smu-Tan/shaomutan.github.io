Thesis project for AI/DS/CL-related Master students

Toward alleviating the off-target issue in Multilingual Neural Machine Translation

Multilingual Neural Machine Translation (MNMT) has shown the potential to conduct zero-shot and even unsupervised translation tasks. Nonetheless, several studies have revealed that MNMT systems tend to generate translations in unexpected or wrong languages (known as the off-target issue), especially in the zero-shot translation scenarios. This project aims to investigate possible reasons and explore potential solutions for off-target issues to improve the zero-shot MNMT.

Thesis project for AI/DS/CL-related Bachelor students

Exploring morphologically-enhanced subword tokenization for low-resource machine translation

The development of subword tokenization methods, such as Byte Pair Encoding, and SentencePiece, help NLP models better cope with the Out-Of-Vocabulary (OOV) problem and alleviate data sparsity issues. However, recent Machine Translation tasks have reported that when facing Morphologically Rich Low-resource Languages (MRLL), existing subword tokenization may not be able to output effective subword units only using limited data. Therefore, in this project, we aim to improve subword tokenization for MRLL, considering their morphological features with the help of linguistic tools. Ultimately, we will validate our method on machine translation-related tasks.

Transfer pre-trained language models to sequence-to-sequence translation models

Neural pre-trained language models (PLMs) such as GPT-3 have demonstrated their superiority in various language generation tasks. Intuitively, transferring a pre-trained language model to a transformer-based Machine Translation (MT) model can be seen as taking advantage of better parameter initialization. The project aims to explore efficient MT model initialization using transfer learning.

Sequential Copying Network for Question-Context Summarization

Conversational Question Answering (ConvQA) is a rising domain aiming to answer human questions in the conversational setting automatically. Under this setting, machines have to refer to the current question and the context in this conversation record (prior QA pairs), as well as possible external knowledge such as Wikipedia. Recent studies have reported that ConvQA models are susceptible to such long input sequences and thus cannot focus on helpful information. This project explores the feasibility of using sequential copying networks for question-context summarization in the ConvQA setting.
